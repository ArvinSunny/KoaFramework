# 开发文档
本项目是基于Koa + Typescript + TypeORM + Winston 搭建

## 项目目录

```
/----
   |---- src
      |---- bin                         项目启动目录
         |---- start.ts                 项目启动文件
      |---- www                         项目基础框架，主要是koa 和 中间件的装配
         |---- KoaFramework.ts          项目入口文件
      |---- entities                    数据库实体类
      |---- modules                     模块
         |---- login                    登录模块
         |---- register                 注册模块
      |---- config                      配置文件夹
         |---- Config                   配置文件
      |---- libs                        公共函数文件夹
         |---- cores                    核心函数
         |---- utils                    工具函数
      |---- middileware                 中间件
      |---- logs                        日志文件夹
      |---- public                      静态资源文件夹
   |---- package.json
   |---- tsconfig.json
   |---- 开发文档.md
```
